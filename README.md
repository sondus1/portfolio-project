# Personal Portfolio Website

A clean, modular personal portfolio built using React and Vite. It showcases my background, key technical skills, real-world development projects, and direct contact details.

##  How to Run the Project Locally

Follow these quick steps to get the project up and running on your local machine:

### 1. Install Dependencies
Before running the app for the first time, you need to install the required package folders (`node_modules`). Open your terminal inside the project folder and run:
```bash
npm install

### 2. Start the Development Server
Once the installation is complete, launch the local development server by running:

Bash
npm run dev

### 3. Open the Application
After running the command, the terminal will give you a local link (usually http://localhost:5173). Control-click or copy-paste that link into your web browser to view your live portfolio!

📁 Project Structure
src/main.jsx — The main entry point of the React application.

src/App.jsx — Controls the layout structure and holds all the sub-components.

src/App.css — Holds all the layout styles and responsive designs.

src/home.jsx, about.jsx, skills.jsx, projects.jsx, contact.jsx — Separate component files representing each section of the website.
